-----------------------------------

**Key Features:**
    - **Interactive CLI**: Scaffold a new Python repository with prompts for repo name, author info, and more
    - **Template-Based**: Uses a pre-configured template to create a ready-to-use project structure
    - **Placeholder Replacement**: Automatically replaces placeholders (repo name, author, email, GitHub username) in all files

-----------------------------------

Support
-----------

*Contributions are welcome!* This library is entirely free. If you find it helpful, please consider starring the repository on [GitHub](https://github.com/erdogant/biscuitcutter).


Installation
-------------

.. code-block:: console

    pip install biscuitcutter


After installation, ``biscuitcutter`` provides an interactive CLI to scaffold a new Python repository:

.. code-block:: console

    biscuitcutter
